Android 環境

Posted on 2016-06-24 Edited on 2020-01-02

參考自圖書館的書

Android Env - Eclipse

下載Java JDK

oracle

下載Eclipse

Eclipse
我裝Eclipse IDE for Java EE Developers

設定Eclipse

功能列 -> Help -> Install New Software
add按鈕
name輸入android
location輸入http://dl-ssl.google.com/android/eclipse/
OK
Developer Tools全選
一直下一步

下載 Android SDK

andriod
下載 android-sdk_r24.4.1-windows.zip 這個
然後安裝執行,放在一個想放的地方

指定 Android SDK

{
功能列 -> Window -> Preference
點左側Android SDK Location選剛剛解壓縮的SDK資料夾(android-sdk-windows)
Apply -> OK
}

Android SDK Manager

功能列 -> Window -> Perspective -> Open Perspective -> 選Java
功能列 -> Window -> Android SDK Manager
選要安裝的SDK
{
Tools所有選項
Android 4.4.2 (API19)所有選項 (用舊的開發)
Extras -> Google USB Driver
}
好了就點選 Install Packages
重開 Eclipse 可以用嚕
以後要安裝不同的版本就點 Android SDK Manager

Android Env - Android Studio

android studio
一直下一步
結束

Android Studio 改字

File -> Settings -> Editor -> Colors&Fonts 記得先按Save as,copy一份存起來

小米手機連不到

到裝置管理員查看連接USB的硬體識別碼,小米應該是2717
在C:\Users\xxxxx\.android\ 下，找到 adb_usb.ini，並寫入 0x2717

到小米大陸網站http://www.mi.com/index.html
服務 -> 資源下載 -> 手機驅動
載完之後 可以在裝置管理員 按更新驅動程式 選剛剛載的那個資料夾

OK

Production - 網站上線前檢測
