macOS 快捷鍵

Posted on 2017-03-03 Edited on 2020-01-02

macOS Keyboard Shortcut

System

關閉螢幕：⌃⇧⏏
休眠：⌥⌘⏏
立即重開機 : ⌃⌘⏏
立即關機 : ⌃⌥⌘⏏

強制結束 : ⌥⌘⎋
登出 : ⇧⌘Q

雙螢幕時 toggle Mirror Displays : ⌘+F1

Global

雙鍵：
⌘+space : Spotlight
⌥+space : iTerm
⌃+space : 切換上一個輸入法

三鍵：
⌥+⌘+space : Finder搜尋 (Global)
⌃+⌘+space : 表情符號
⌃+⌥+space : 切換下一個輸入法

Finder

⌘⇧. : toggle顯示隱藏檔案
⌘[ ⌘] : 上一頁、下一頁
⇧⌘D : 跳到Desktop
⇧⌘R : 跳到AirDrop
⇧⌘F : 跳到All My Files

⌘1, 2, 3, 4 : View as Icons, List, Columns 👍, Cover Flow
⌃⌘1, ⌃⌘2 : Arrange By (Name, Kind) 👍
⌃⌘4 : Arrange By Data Added
⌃⌘3, ⌃⌘5 : Arrange By (Data Last Opened, Data Modified) 👍
⌃⌘6 : Arrange By Size 👍
記：⌘3, ⌃⌘3

Apps

⌘+H : 隱藏APP
⌘+M : 縮到右下
⌥+⌘+H : 隱藏除了自己的APP

Browser

⌘+T : 開新分頁
⌘+W : 關閉分頁
⌘+⇧+T : 開上一個關閉的分頁
⌘+L : 跳到搜尋列
⌘+⌥+→ : 左右分頁跳跳
⌃+tab, ⌃+⇧+tab : 下/上一分頁
⌘+(1..8) : 跳數字頁
⌘+9 : 跳最後一頁
⌘+` : 兩個不同的browser切換（未全屏）
⇧⌘H : 按home button

⌘+⇧+B : 顯示書籤列
⌘+⇧+F : 全螢幕時顯示工具列

Text Editor

⌃+A : 跳到文字開頭
⌘+左or右 : 跳到文字開頭或結尾 (文字編輯器才能用)
⌃+K : 刪除游標後的文字
⇧+左or右 : 一般選字
⇧+⌃+左or右 : 單詞選字
⇧+⌥+左or右 : 單詞選字
⇧+⌘+左or右 : 屌選字

Terminal

⌃A : 跳到command開頭
⌃E : 跳到command結尾
⌥+Click : 跳到滑鼠點擊command
⌃W : 刪除最後一個單字
⌃U : 刪除游標之前的command
⌃K : 刪除游標之後的command
⌃R : 搜尋command
⌃L : clear (iTerm: ⌘+R)
⌘K : clear(滑不回去)
~. 可以直接退出一個session，沒用的話先按Enter一下。

iTerm2

DOC: https://www.iterm2.com/documentation-one-page.html
⌘D Splits Vertically
⌘⇧D Splits Horizontally
⌘[ ⌘] Next/Previous Pane

Preview

  • ⌥⌘1, ⌥⌘2 : Hide Sidebar, Thumbnails
  • ⌘1, ⌘2, ⌘3 : Continuous Scroll, Single Page, Two pages
  • ⌘9, ⌘0, ⌘+/- : Zoom to Fit, Actual Size, Zoom In/Out
  • ⇧⌘A : Show Markup Toolbar
  • ⌥⌘G : Go to Page
  • ` : Show Magnifier

雜

在桌面上按 ⇧+⌘+delete 可以清空垃圾桶
⇧⌥+音量 ： 調四分之一音量
⌃+fn+⇧+F2：Active menu bar
